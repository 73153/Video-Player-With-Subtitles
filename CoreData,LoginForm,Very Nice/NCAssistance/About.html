NCA v1.0
NCA is a keychain storing software that is designated to help you save your accounts logins, passwords and some additional information. The software is protected by an access code that you set on initial login. A security question is also stored, in case the access code is forgotten.
NCA do NOT access any personal information on your device. Nor does NCA send the account information you stored to any servers.
For security reasons, please do NOT save critical information such as bank account password or social security number in any electronic form, including saving in NCA.
