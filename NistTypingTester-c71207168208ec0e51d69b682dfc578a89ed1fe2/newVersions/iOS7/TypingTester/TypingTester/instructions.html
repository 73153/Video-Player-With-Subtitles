Memorization

On the following screens you will be presented with a string to memorize. The strings will vary in length and compostion. Please memorize the strings as quickly as possible. The memorization process will take place over three screens, you may navigate between the screens using the '<' and '>' buttons if you need to.

  • Free Practice - This screen displays the string and provides a text area that you may use as a work area if needed
  • Practice - This screen displays the string and provides a text box to enter the string in. You must enter the string correctly a minimum number of times to move on.
  • Verify - This screen does not display the string, and you must enter the memorized string correctly to proceed. After successful verification you will proceed to the entry phase.

Entry

After completing the memorize phase you will be asked to enter the string you have memorize a number of times. Please enter the string as quickly and accurately as possible. Once you have entered the string press the '>' to proceed.

Repetition

The memorize and repetition phases will repeat a number of times.
